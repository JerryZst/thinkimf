# ThinkIMF是什么
## 一款基于PHP7+ SWOOLE + MariaDb/Mysql 的物联网设备管理框架
为物联网3.0进行驱动的软件,跨平台，开放源代码

* 用户管理
* 扫码登录
* 权限管理
* 物联网设备管理 
* 物联网设备通信
